# 🤖 Hermes Agent Tryout Demo

A complete setup to try out **Hermes Agent** — the self-improving AI agent by Nous Research.

## 🚀 Quick Start

```bash
# Clone this repo
git clone https://github.com/enrikaclever/refactored-potato.git
cd refactored-potato

# Run the installer
bash hermes-setup.sh

# Verify installation
bash quick-test.sh

# Start chatting!
hermes
```
